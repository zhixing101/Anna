## Z025/11/01（六）

四川全兴足球队是中国足球职业化初期一支极具影响力的球队，它承载了一代人的记忆，其兴衰史也是中国足球发展的一个缩影。

### ⚽ 成立与早期发展 (1993年以前)

四川全兴队的前身是四川省足球队。在中国足球迈向职业化的前夜，这支队伍为后来的故事埋下了伏笔。

### 🏆 甲A时代的辉煌与坎坷 (1994-2001)

1994年，四川全兴足球俱乐部正式成立，并作为创始成员参加了首届甲A联赛。这支球队迅速以其鲜明的特色风靡全国：

*   **“黄色狂飙”与“金牌球市”**：球队亮黄色的战袍和成都主场山呼海啸般的助威声，共同构成了名震全国的“黄色狂飙”。成都也因此获得了“金牌球市”的美誉，一票难求的景象屡见不鲜。
*   **“成都保卫战”**：1995年，四川全兴队一度深陷降级区，在最后两轮必须全胜才能保级的绝境下，引发了全国关注的“成都保卫战”。最终球队成功保级，这场惊心动魄的战役成为中国足球职业化初期火爆场景的经典注脚。
*   **战绩与球星**：在**余东风**的率领下，球队拥有**魏群**（打进了中国职业联赛的第一粒进球）、**马明宇**（曾南下加盟广东宏远，后回归）、**姚夏**、**邹侑根**等一批极具人气的本土球星。在1999年和2000年，四川全兴队连续两年获得甲A联赛季军，达到队史巅峰。
*   **文化贡献**：四川球迷创造的助威口号“**雄起**”以及“下课”等词汇，从成都球场传遍大江南北，极大地丰富了中国的足球文化。

### 🔄 所有权变更与动荡 (2002-2005)

2001年底，全兴集团宣布退出中国足坛。此后，球队进入了一段动荡时期：
*   2002年，球队被**大连实德集团**收购，更名为四川大河俱乐部。
*   2003年，球队更名为**四川冠城**，并成为中超联赛的创始球队之一。
*   这一时期，俱乐部陷入资金短缺、管理混乱以及“实德系”的关联关系纠纷中。

### 😔 解散与后续 (2006年及以后)

由于四川省足球协会无法与大连实德在规定期限内达成一致，**四川冠城俱乐部于2006年1月27日正式解散**，这支从甲A时代走来的球队就此退出历史舞台。

尽管如此，四川全兴留下的足球火种与文化基因并未完全熄灭。近年来，以**成都蓉城**为代表的新生力量在中超联赛中表现出色，再次点燃了四川球迷的热情。同时，面向草根球员的**四川省城市足球联赛（“川超”）** 也应运而生，许多像魏群、姚夏这样的“老全兴”成员也参与其中，继续为本地足球贡献力量。

### 📊 历史评价与影响

四川全兴足球队的历史地位和影响，可以通过下表清晰地概览：

| **维度** | **评价与影响** |
| :--- | :--- |
| **足球文化** | 开创了极具感染力的"金牌球市"，将具有地方特色的"雄起"等助威方式推广至全国。 |
| **球迷情感** | 是上世纪90年代一代四川乃至全国球迷的青春记忆和情感寄托，代表了甲A联赛的激情岁月。 |
| **商业运营** | 全兴集团的"投资-广告效应-退出"模式，揭示了早期中国足球俱乐部在商业可持续性方面面临的挑战。 |
| **历史教训** | 俱乐部的最终解散，暴露出中国足球在职业化初期在俱乐部股权结构、关联关系和长期规划方面的不成熟。 |

### 💎 总结

四川全兴队的故事，是一段关于**激情、归属与城市荣耀**的传奇。它完美地诠释了足球如何与一座城市的精神文化深深绑定。虽然俱乐部因种种原因最终落幕，但它所播下的足球火种和创造的黄色记忆，至今仍在四川延续。
