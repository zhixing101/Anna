根据您的问题，目前日本正在进行或即将开展多项与**降低眼压相关的眼科药物临床试验**。以下是针对“最新降压眼药水”受试计划的详细介绍。

## 一、日本正在开展的降压眼药水受试计划

### 1. NCX 470（bimatoprost grenod）——Ⅲ期临床试验

这是目前**最新、最受关注**的降压眼药水之一。NCX 470是一种新型一氧化氮供体型贝美前列素滴眼液，通过增加房水流出通路来降低眼压。

| 项目 | 内容 |
|------|------|
| **试验药物** | NCX 470 (bimatoprost grenod) |
| **适应症** | 开角型青光眼、高眼压症 |
| **试验阶段** | Ⅲ期临床试验 |
| **启动时间** | 2025年夏季在日本启动  |
| **申办方** | Kowa（兴和株式会社，与法国Nicox公司合作） |
| **特点** | Ⅲ期数据显示眼压降幅最高可达10mmHg，美国NDA预计2026年夏季提交  |

**参与方式**：该试验目前正在日本全国多家医疗机构招募参与者。如需了解具体参与医院和招募条件，建议咨询主治医生或关注各医院的临床试验招募信息。


### 2. Ripasudil（利舒地尔/Glanatec）——相关临床试验

Ripasudil是日本已上市（2014年获批）的ROCK激酶抑制剂类降眼压药。目前仍在进行多项临床试验，探索其新适应症。

| 项目 | 内容 |
|------|------|
| **试验药物** | Ripasudil 0.4%滴眼液 (Glanatec®) |
| **正在开展的试验** | ① Fuchs角膜内皮营养不良患者的角膜保护研究（Ⅲ期）<br>② 白内障术后角膜内皮功能保护研究<br>③ 视网膜色素变性相关研究 |
| **试验阶段** | Ⅱ/Ⅲ期 |
| **实施地区** | 日本、美国、德国等  |

**最新动态**：2025年，AI系统Robin在2.5个月内自主发现Ripasudil可作为干性年龄相关性黄斑变性的新候选药物，相关研究正在推进中 。


### 3. Sepetaprost——已获批上市的新药

| 项目 | 内容 |
|------|------|
| **药物名称** | Sepetaprost（DE-126/ONO-9054） |
| **作用机制** | EP3受体激动剂 + 前列腺素FP受体激动剂（双靶点） |
| **获批时间** | 2025年8月25日在日本获批上市  |
| **适应症** | 青光眼、高眼压症 |
| **特点** | 新型双重激动剂，作用机制不同于现有药物 |

**说明**：此药已于2025年获批上市，相关临床试验已经完成，不再招募受试者。


### 4. Avacincaptad Pegol——眼内注射药物（非滴眼液）

| 项目 | 内容 |
|------|------|
| **药物名称** | Avacincaptad Pegol (ASP3021) |
| **适应症** | 年龄相关性黄斑变性继发的地理萎缩 |
| **给药方式** | 玻璃体内注射（非滴眼液） |
| **试验阶段** | Ⅲ/Ⅳ期 |
| **计划招募人数** | 20人 |
| **试验状态** | 进行中（预计2027年6月完成）  |

**说明**：此试验为注射剂型，非滴眼液，但属于眼科降压相关领域。


## 二、如何查询和参与这些受试计划

### 1. 主要查询渠道

| 渠道 | 说明 |
|------|------|
| **主治医生咨询** | 最直接的方式。向眼科主治医生表明参与临床试验的意愿，医生会根据您的病情判断是否符合条件 |
| **各医院临床研究中心** | 如圣隶浜松医院临床研究中心等，会公示招募中的试验信息  |
| **jRCT（Japan Registry of Clinical Trials）** | 日本临床试验注册中心，可查询官方登记的试验信息 |
| **UMIN-CTR** | 日本大学医学信息网络临床试验注册中心  |

### 2. 参与流程

根据日本临床试验的一般流程，参与受试计划通常包括以下步骤 ：

1. **寻找试验信息**：在医院网站、公告栏或通过医生了解正在招募的试验
2. **初步筛选**：确认是否符合基本入选条件（年龄、疾病类型、治疗史等）
3. **知情同意（インフォームドコンセント）**：医生和临床试验协调员详细说明试验目的、方法、来院日程、检查内容、预期效果及可能出现的副作用
4. **签署同意书**：充分理解并自愿同意后签署
5. **参加试验**：按照方案接受治疗和定期检查

### 3. 参与注意事项

- **所有临床试验均为自愿参加**，可随时退出
- **入选有严格标准**：即使疾病相符，也可能因其他因素（如既往治疗史、合并症等）不符合条件
- **名额有限**：即使招募期内，达到目标人数后也会停止招募 
- **需要与主治医生商量后咨询**
