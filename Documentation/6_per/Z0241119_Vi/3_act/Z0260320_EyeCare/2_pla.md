# 中山医学大学全面介绍

## 一、学校概况

中山医学大学（Chung Shan Medical University，简称“中山医”“CSMU”）是位于台湾台中市的一所私立医学大学。学校由周汝川博士于1960年创立，最初仅设牙科，名为“中山牙科专科学校”，后于2001年正式升格为大学。

| 项目 | 内容 |
|------|------|
| **创办时间** | 1960年 |
| **学校类型** | 私立医学大学 |
| **校训** | 诚、爱、精、勤 |
| **现任校长** | 黄建宁 |
| **学生人数** | 约7,085人（2023年） |
| **教师人数** | 692人（2023年） |
| **校区地址** | 台中市南区建国北路一段110号（邻近捷运大庆站、台铁大庆站） |

学校设有医学院、口腔医学院、医学科技学院、健康管理学院、护理学院、医学人文暨社会学院等六个学院，其附设医院为台中市三大医学中心之一。

## 三、优秀学科

中山医大的学科以医学相关领域见长。学校设有医学院、口腔医学院、医学科技学院、健康管理学院四大核心学院，涵盖医学、牙医、护理、医技、视光、语言治疗与听力学、职安、营养、医管等多元领域。

**附设医院特色**：学校附设医院分为大庆院区、中港院区、太原院区及文心院区四大院区，其中大庆院区规模最大，为台中市三大医学中心之一。医院已引进正子断层扫描仪、回旋加速器、螺旋光子刀等高科技诊疗设备。

## 五、海外在职学制

中山医学大学设有**国际健康产业经营管理硕士在职专班**，旨在培养健康产业的中高阶经营管理人才，课程涵盖医疗管理、职业安全卫生、健康产业与外文应用等方向。

### 澳门境外硕士专班

中山医大在澳门设有**医学检验暨生物技术学系境外硕士在职专班**（医学检验技术理学硕士学位课程），与澳门理工学院合作开设。

| 项目 | 内容 |
|------|------|
| **合作单位** | 澳门理工学院（教学场所） |
| **课程期限** | 至少2年，兼读制 |
| **毕业学分** | 24学分（含必修12学分、选修8学分、硕士论文6学分） |
| **授课形式** | 面授 |
| **开设年份** | 2016年8月 |

该课程经澳门特别行政区政府确认，为当地带来效益并许可运作。台湾台中校本部亦设有推广教育处，提供在职进修课程。
